# PPG Global Paint Scraper

A Python automation bot that scrapes automotive paint data from PPG for different regions including Europe, Asia, America, and Australia.  
The scraper collects car paint information and exports the extracted data into CSV files for further analysis or usage.

## Features

- Scrapes car paint data from multiple global regions
- Automated browser interaction using Playwright
- Structured data export to CSV
- Logging system for monitoring and debugging
- Clean and organized data handling with Pandas

## Technologies Used

- Python
- Playwright
- Pandas
- CSV
- OS
- Logging
- Datetime

## Installation

1. Clone the repository

bash
git clone https://github.com/Mujahid1214/ppg-global-paint-scraper.git
cd ppg-global-paint-scraper
2
pip install -r requirements.txt
3
playwright install
